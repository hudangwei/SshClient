Freeze Mode

Freeze mode is entered by clicking the Android's menu button then clicking on Freeze. While in freeze mode, data from the host is blocked so the screen won't update. But you can still scroll around. Also, the keyboard will not send characters to the host. Rather, there are special characters that can be used to copy data to a clipboard:

  • b - begin text selection at the cursor
  • c - copy text selection to a clipboard
  • i - move cursor up one line
  • j - move cursor left one character
  • k - move cursor down one line
  • l - move cursor right one line
  • r - reset (discard) selection

To exit freeze mode, click the Freeze button again. The screen colors will go back to normal and the host will be released to update the screen.
